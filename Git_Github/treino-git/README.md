# 📝 Pasta Treino – Git & GitHub

Esta pasta contém **exercícios práticos** para treinar comandos e fluxos do Git e GitHub.  
O objetivo é praticar de forma isolada, sem impactar os arquivos principais do repositório.

---

## 📂 Estrutura

- `.gitignore` → arquivo para praticar o controle do que deve/ não deve ser versionado.  
- `my-tasks.txt` → lista de tarefas usada para treinar commits, branches, merges e stashes.  
- `README.md` → explicação da pasta e dos exercícios.  

---

## 🎯 Exercícios já praticados

- Criar e alternar **branches**
- Usar **stash** para guardar alterações temporárias
- Trabalhar com **merge** e **rebase**
- Criar e listar **tags**
- Alterar o conteúdo de arquivos e acompanhar o histórico de commits
